# Architecture Decision Records

This directory records significant, cross-cutting platform decisions. Each ADR is immutable once **Accepted** — to change a decision, add a new ADR that supersedes the old one (and update the old one's status to `Superseded by ADR-XXXX`).

| ADR | Title | Status |
|---|---|---|
| [0001](0001-release-automation-strategy.md) | Release Automation Strategy | Accepted |
| [0002](0002-github-app-identity-strategy.md) | GitHub App Identity Strategy | Accepted |

## Format

Lightweight [MADR](https://adr.github.io/madr/)-style: **Context → Decision Drivers → Considered Options → Decision → Consequences**. Keep them short and decision-focused; link to tickets and primary sources rather than restating them.
